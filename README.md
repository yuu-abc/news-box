# 分析メモ付きニュースボード

ニュースを、要約・分析・影響メモ・自分用メモつきのカードとして整理できるシングルページアプリです。ブラウザだけで動き、登録データは `localStorage` に保存されます。HTTPSまたはlocalhostで一度開くと、対応ブラウザではオフラインでも再表示できます。

## 主な機能

- ChatGPTなどから受け取ったJSONニュースの貼り付け、検証、整形
- タイトル、カテゴリ、重要度、信頼度、ステータス、取込日、お気に入りによる一覧管理
- 要約、背景、分析、影響メモ、自分用メモ、不確実な点、次に見るべきこと、参考の表示
- ニュースカードへの画像の手動アップロードと保存
- ボード画面で件数・重要ニュースを俯瞰
- お気に入り登録、ニュース単体の削除、ニュース単体のJSONコピー
- 検索とフィルタリング
- 登録時に自動で取込日を記録し、一覧と詳細に表示
- Service WorkerとWeb App Manifestによるオフライン起動・インストール対応

## 使い方

1. ローカル確認では `python3 -m http.server` などで配信し、ブラウザで `http://localhost:8000/` を開きます。
2. 「プロンプトをコピー」からニュース整形用プロンプトをコピーします。
3. ChatGPTなどで生成したJSONを「JSON貼り付け」に入れます。
4. 必要に応じてプレビュー画面で編集し、画像を残したい場合は「画像」の箱から手動でアップロードします。
5. 「登録」を押します。
6. 「ボード」または「詳細」で登録済みニュースを確認します。
7. 一度読み込んだ後は、通信がない状態でも同じURLからアプリを開けます。
